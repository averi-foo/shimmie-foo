# shimmie-foo
This is a repository for custom shimmie extensions and themes intended for the website. I chose this folder structure over directly forking shimmie for less hassle in setup and development.

## Installation
``git clone https://github.com/averi-foo/shimmie-foo``

Git clone anywhere on your system, but preferably in the same folder as your shimmie2 folder is in (next to shimmie2, not inside of shimmie2).

Then create symlinks for shimmie-foo extensions and themes and configs inside of your shimmie2 folder, or run the install script to do it for you (wip).
